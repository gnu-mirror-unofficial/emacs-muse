Emacs Muse

Emacs Muse (también conocido como "Muse" o "Emacs-Muse") es un entorno de autor y publicación para Emacs. Simplifica el proceso de escritura de documentos y su publicación en varios formatos de salida.

Obtención de Emacs Muse

Muse se desarrolla con git. Las instrucciones de cómo obtener la última versión de desarrollo, consúltese el capítulo pertinente del Manual de Muse.

La última versión de desarrollo también se puede descargar en GitHub.

Las versiones estables están en la zona de descargas de la página de proyecto de Muse-el en GNA.

También hay paquetes de Muse para Debian y Ubuntu. Se llaman muse-el. Véase la página de los Paquetes Debian para leer las instrucciones acerca del uso de los repositorios de paquetes Debian y Ubuntu del anterior mantenedor.

Documentación

La documentación de Emacs Muse está en línea, como la de la documentación de la mayoría del software GNU. También se puede obtener más información acerca de Emacs Muse ejecutando info muse.

Listas de correo

Para reportar problemas, pedir ayuda, conversar acerca de asuntos diversos en cuanto al mundo de la publicación (entre otras cosas), mantenerse informado de los anuncios o seguir el desarrollo de Muse, hay disponibles unas cuantas listas de correo. Como medida de prevención contra el spam, primero hay que suscribirse a la lista de correo antes de poder enviar mensajes a ella.

muse-el-announce@gna.org
Es una lista de tráfico bajo para los anuncios relativos a Muse.
URL: http://mail.gna.org/listinfo/muse-el-announce/
Gmane: gmane.emacs.muse.announce
muse-el-discuss@gna.org
Debates, correcciones de fallos, sugerencias, consejos y asuntos similares para Muse. Esta lista de correo también incluye el contenido de la lista de Anuncios.
URL: http://mail.gna.org/listinfo/muse-el-discuss/
Gmane: gmane.emacs.muse.general
muse-el-logs@gna.org
Mensajes de registro (Log) para las compulsas de código realizadas a Muse.
URL: http://mail.gna.org/listinfo/muse-el-logs/
Gmane: gmane.emacs.muse.scm
muse-el-commits@gna.org
Informes de fallos generados para Emacs Muse.
URL: http://mail.gna.org/listinfo/muse-el-commits/
Gmane: gmane.emacs.muse.cvs
muse-el-internationalization@gna.org
Debates sobre la traducción del sitio web de the y de su documentación en muchos idiomas.
URL: http://mail.gna.org/listinfo/muse-el-internationalization/
Gmane: gmane.emacs.muse.internationalization

El sitio web de Gmane permite al usuario acceder a las listas de correo de varias útiles maneras, tales como: canales RSS, una interfaz de estilo blog y como grupo de noticias.

Desarrollo

Para infomar de un fallo o sugerir un cambio, es conveniente revisar el Registro de fallos. Todos los fallos enviados a este registro aparecerán igualmente en la lista de correo muse-el-commits.

Mantenedor

El mantenedor actual de Emacs Muse es Alex Ott. Para contactar con él, por favor, es preferible usar las listas de correo.
